# Module 10 — Security

Security is non-negotiable. This module covers the key security topics that appear in senior iOS interviews.

## Key Interview Themes

- **Keychain vs UserDefaults** — always Keychain for sensitive data
- **Certificate pinning** — implementation and operational tradeoffs
- **ATS** — what it enforces and when exceptions are needed
- **Token security** — storage, transmission, expiry
- **Jailbreak detection** — awareness and limitations
