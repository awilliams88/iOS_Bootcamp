# Debugging and Performance Tools

## Instruments

Use for profiling runtime behavior.

Key tools:
- Time Profiler
- Allocations
- Leaks
- Network

## Memory Graph Debugger

Useful for retain cycle detection.

## Performance Areas

- startup time
- scrolling smoothness
- memory growth
- SwiftUI recomputation
- networking bottlenecks

## Interview Answer

I use profiling tools rather than guessing, especially Time Profiler, Leaks, and Memory Graph for performance and memory investigations.
