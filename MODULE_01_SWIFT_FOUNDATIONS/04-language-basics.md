# Swift Language Basics

## Variables

```swift
let constant = 10
var mutable = 20
```

Use `let` by default.

## Type Inference

```swift
let name = "Arpit"
```

Compiler infers type.

## Explicit Types

```swift
let age: Int = 30
```

## Tuples

```swift
let user = (id: 1, name: "Arpit")
```
