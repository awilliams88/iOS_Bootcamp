# Swift Quick Revision Cheatsheet

## Rapid Recall

### Value vs Reference
Struct = value
Class = reference
Actor = reference + isolation

### ARC
Classes and actors only.

### weak vs unowned
weak = optional + auto nil
unowned = non-optional + crash if invalid

### Closures
Capture surrounding context.

### Protocols
Behavior contracts.

### Generics
Reusable type-safe abstraction.

### Error Handling
throws / try / Result

### Concurrency
async/await + actors + MainActor
